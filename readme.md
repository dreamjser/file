# @dreamjser/copy-file
copy file into target directory

## Installation

```
    npm install @dreamjser/copy-file

```

## nodejs

```
    const file = require('@dreamjser/file');
```

## document

```
    // mkdir
    file.mkdir('dest', () => {});

    // copyfile
    file.copy('src', 'dist', () => {})
```
